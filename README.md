# 🚚 لوحة توزيع الهدايا - Delivery Dashboard Pro

لوحة تحكم احترافية متقدمة لإدارة توزيع الهدايا على المستخدمين مع واجهة حديثة وميزات متطورة.

## 📁 بنية المشروع
delivery-dashboard/
├── index.html # الصفحة الرئيسية
├── css/
│ ├── main.css # الأنماط الأساسية
│ ├── dashboard.css # لوحة التحكم
│ ├── map.css # الخريطة
│ ├── modal.css # النوافذ المنبثقة
│ └── responsive.css # التجاوب
├── js/
│ ├── firebase-config.js # إعدادات Firebase
│ ├── data-manager.js # إدارة البيانات
│ ├── users-module.js # وحدة المستخدمين
│ ├── map-module.js # وحدة الخريطة
│ ├── delivery-module.js # وحدة التوصيل
│ ├── stats-module.js # وحدة الإحصائيات
│ ├── ui-utils.js # أدوات UI
│ └── app.js # التطبيق الرئيسي
└── README.md

## 🎯 المميزات الرئيسية

### 📊 لوحة تحكم شاملة
- إحصائيات مباشرة (6 بطاقات)
- رسوم بيانية تفاعلية
- آخر النشاطات
- إجراءات سريعة

### 👥 إدارة المستخدمين
- جدول تفاعلي مع فرز وفلترة
- بحث متقدم
- عرض تفاصيل شاملة
- تصدير إلى CSV/JSON
- نسخ معلومات المستخدم

### 🗺️ خريطة تفاعلية
- عرض مواقع المستخدمين
- علامات ملونة حسب الحالة
- نوافذ منبثقة غنية بالمعلومات
- تكبير/تصغير
- فتح في Google Maps

### 📦 إدارة التوصيلات
- إنشاء طلبات جديدة
- تتبع الحالات (معلق/تم/ملغي)
- فتح الموقع في Google Maps
- تسجيل التسليم

### 📈 تحليلات متقدمة
- توزيع الأجهزة
- توزيع المتصفحات
- توزيع الدول
- أوقات النشاط
- أنظمة التشغيل
- أنواع الاتصال

### ⚙️ إعدادات
- وضع ليلي/نهاري
- تغيير سرعة التحديث
- الإشعارات الصوتية
- تصدير البيانات
- مسح البيانات المحلية
